## Listas

As listas são uma coleção de itens, que podem armazenar quaisquer informações.

### Criando uma lista
Nos podemos criar uma lista da seguinte forma:

```py
lista = ['dado1', 'dado2', 'dado3']
```

### Obtendo os valores de uma lista
Para obter os valores presentes na lista podemos usar o print.

```py
lista = ['dado1', 'dado2', 'dado3']
print(lista)
```

### Obtendo um valor especifico
Para obter um valor especifico dentro de uma lista podemos fazer o seguinte:

```py
lista = ['dado1', 'dado2', 'dado3']
print(lista[0])
```

Obs: a primeira posição do elemento é 0 a segunda 1 e assim por diante
